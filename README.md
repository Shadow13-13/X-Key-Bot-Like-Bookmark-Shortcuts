# X Key Bot — Like & Bookmark Shortcuts

Custom hotkeys for **liking/unliking** and **bookmarking/unbookmarking** posts on X (Twitter).

- Press your chosen key on the post nearest to the center of your screen
- Draggable button that snaps to any screen edge
- Fully customizable hotkeys (supports Ctrl/Alt/Shift/Meta combos)

## Installation

1. Install **Tampermonkey** (Chrome/Edge/Firefox) or **Violentmonkey**
2. Click this link to install:

   **[Install X Key Bot](https://raw.githubusercontent.com/YOUR-USERNAME/x-key-bot/main/x-key-bot.user.js)**

3. Done! The ⭐ button will appear on X.

## Default Hotkeys
- `L` → Like / Unlike
- `B` → Bookmark / Remove bookmark

You can change them anytime by clicking the floating button.

## Features
- Works on both `x.com` and `twitter.com`
- Smart "nearest post" detection
- Beautiful draggable + snap UI
- Toast notifications
- Remembers your settings and button position
